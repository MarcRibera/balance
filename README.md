# Balance Project
Project to visualize financial data

Technologies:
- vue 2 + element ui
- Highcharts
- csv-loader



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
